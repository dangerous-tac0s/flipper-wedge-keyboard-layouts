# Dvorak keyboard layout
# Alternative ergonomic layout
# Numbers are in the same positions as QWERTY
# Letters are completely rearranged
#
# Dvorak letter positions:
# Top row:    ' , . p y f g c r l
# Home row:   a o e u i d h t n s
# Bottom row: ; q j k x b m w v z
#
# HID keycodes for hex letters on Dvorak:
# A = position of 'a' key (home row, left pinky) = US 'a' position = 4
# B = position of 'b' key (bottom row) = US 'n' position = 17
# C = position of 'c' key (top row) = US 'i' position = 12
# D = position of 'd' key (home row) = US 'e' position = 8
# E = position of 'e' key (home row) = US 'd' position = 7
# F = position of 'f' key (top row) = US 't' position = 23
